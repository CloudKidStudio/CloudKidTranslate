load

(
  • dict
  • [callback]
)
Translate static

Defined in load:61

Load the full dictionary containing all translations, this can also be used to load separate JSON files which contain the translation. Each JSON file must contain a key matching the locale which to use;

// Load using JSON
Translate.load("lang.json", function(){ 
    // Finished loading
});

// or load the object directly
var dict = {
    "en" : {
        "title" : "My Site"
    }
};
Translate.load(dict);

Parameters:

  • dict Object | String

    The translation dictionary or file path to the translation dictionary

  • [callback] Function optional

    The methond to callback if we're loading a file
